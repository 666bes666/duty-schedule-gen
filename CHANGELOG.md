## [3.0.0] - 2026-03-22

Финальный релиз перед переходом в режим обслуживания (только багфиксы).

### Ключевые изменения

- **PostProcessStage protocol + Pipeline** — постпроцессинг декомпозирован в типизированные стадии с метриками качества (ScheduleSnapshot)
- **Multi-employee pairwise swap** — парные свопы между сотрудниками для оптимизации изолированных выходных
- **3-way ротация вечерних смен** — предотвращение концентрации вечерних смен у одного сотрудника
- **Адаптивный пайплайн для малых команд** (≤4 дежурных) — дополнительные стадии балансировки
- **OptimizationPriority** — выбор приоритета оптимизации (изолированные выходные / вечерние смены / серии / выходные) с UI-селектором
- **Валидация расписания** после постпроцессинга (hard/soft constraint violations)
- **Graceful degradation WeasyPrint** — PDF-экспорт работает даже без установленного WeasyPrint
- **REST API** с аутентификацией, rate-limiting и what-if эндпоинтами (FastAPI)
- **CP-SAT solver** — опциональный оптимальный солвер через Google OR-Tools
- **Многомесячная генерация** с автоматическим carry-over между месяцами
- **Трёхуровневый CI/CD** (dev → test → main) с auto-tag и GitHub Releases
- **Аудит логирования** — тайминги, трейсинг, консистентность по всем модулям
- **Безопасность** — санитизация экспортов, non-root Docker, Tornado 6.5.5 (CVE-2026-31958)
- Документация полностью актуализирована под состояние кода
